# Personal Introduction Program
## Project Overview
This is a simple Python program that asks the user for their name, age, and hobby. After collecting the information, the program displays a friendly welcome message containing the user's details.

## Project objectives
- Learn how to use the input() function.
- Learn how to store data in variables.
- Learn how to display output using print().
- Understand basic Python programming concepts.
## Setup and Installation Instructions 
### Requirements
- Python 3.14.5
## How to Run:
- Download the repository
- Open terminal
- Navigate to the project folder
- Run:

  python personal_intro.py

## Code Explanation
The program uses:

- input() to collect information from the user.
- Variables (name, age, and hobby) to store user data.
- print() statements to display a personalized welcome message.
